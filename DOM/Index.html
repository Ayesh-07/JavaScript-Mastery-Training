The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document
as a tree of objects, where each object corresponds to a part of the document, such as elements, attributes, and text.

Structure:

The DOM represents a web page as a hierarchical tree structure. The document starts with a document object, which contains
a documentElement representing the root of the HTML or XML document (usually the tag in an HTML document).

Nodes:

Each part of the document (elements, attributes, text) is represented as a node in the tree.
There are several types of nodes:

  • Element nodes: Represent HTML or XML tags (e.g., div, p).
  • Attribute nodes: Represent attributes of elements (e.g., class, id).
  • Text nodes: Represent the text inside elements.

Manipulation:

Using JavaScript, you can interact with and manipulate the DOM. For example, you can:

  • Change the content of elements.
  • Add or remove elements.
  • Modify attributes.

Events:

The DOM also handles events, like user interactions (clicks, form submissions) and changes to the document. JavaScript can listen
for and respond to these events.
